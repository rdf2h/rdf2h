More examples

  • https://github.com/zazukoians/trifid-ld/blob/master/data/public/rdf2h/matchers.ttl: Provides some generic matchers to render any resource.
  • https://github.com/zazukoians/municipality-browser/blob/gh-pages/agepyramid-matchers.ttl : Matchers that render an Age Pyramid with d3.js.
  • https://github.com/zazuko/lindas-datasets/blob/gh-pages/index.html : Contains matchers to render datasets descriptions.
